# in.txt:
* 'b1', 'b2' - inititial and final numbers (2..16) of numeral systems
* 'number in b1'
# out.txt:
* 'wrong input' - if input data is wrong
    or
* 'number in b2'

# Examples
## in.txt
    4 8
    243

## out.txt
    bad input
    
## in.txt
    5 8
    243

## out.txt
    111

## in.txt
    16 13
    ABbAc.54F

## out.txt
    1b8220.440c2300
