Calculator (executes string with math expression).

# in.txt:
* 'expression' - string of '+', '-', '*', '/', braskets and integer numbers
# out.txt:
* 'syntax error' or 
* 'division by zero' or
* 'execution result'

# Examples
## in.txt
    5+4*3-5/(2+2*2)-8*5-5

## out.txt
    -30

## in.txt
    1()

## out.txt
    syntax error
